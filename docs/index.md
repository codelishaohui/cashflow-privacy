4. 当前订阅周期的费用，我们无法为您办理退款，请务必在付款时考虑清楚。

### 隐私政策

钱去哪了App，使用CloudKit实现了您的数据备份功能，而CloudKit是Apple提供的API，所以您的数据只存在您的设备和您的iCloud账户中。
如果您没有在钱去哪了App中启用iCloud同步，则您的数据只存在您的设备中。
